## 什么是书源？

简单来说，书源就是扩展书籍获取的渠道。

大部分阅读器还是以本地书籍阅读为主，但对于某些文体（比如连载的网络小说），由于每日高强度的更新，本地书籍作为载体就不太合适了。

> *这位读者，你也不想每天都导入好几遍 txt 吧？*

这样一来，在线书源就应运而生。它可以收集多个在线书籍来源，并在本地阅读，可以随时进行更新以获取最新章节，降低追更难度，避免读者在几个 app 中奔波。

书源需要手动编写，但我猜大部分用户应该不太会用编程方式来写书源。如果你喜欢阅读网文，那么我这里有一个 “热心网友” 写的网文书源：

- [下载地址](https://github.com/Richasy/ReaderCopilot.Public/releases)

:::tip

该书源基于 .NET 7 编写，使用前请确保你的设备上安装了相关的运行环境。

可以在 [下载 .NET 7.0](https://dotnet.microsoft.com/zh-cn/download/dotnet/7.0) 中下载对应平台的 SDK 或 运行时。

:::

:::warning

阅读助理提供书源插件用于扩展功能，但是本身不提供书源制作服务，文中提及的书源插件仅供测试哟~

:::

## 怎么导入？

书源插件是一个后缀为 `.rcpkg` 的一个文件，实际上你把后缀名改成 `.zip` 后就能查看里面的内容了。

嗯……就是这么简单粗暴。

现在，进入应用的设置界面，你能看到一个 `书源插件设置`。

<img src={pluginSetting} width="380px" />

点击 `导入书源插件`，选择刚刚下载的 `.rcpkg` 文件导入即可。

导入完成后，你可以在 `书源插件` 里切换当前已有的插件。

## 怎么使用？

一个插件里包含一个或多个书源，你可以进入 `网文浏览` 界面，切换不同的书源，浏览不同的信息，或者搜索书籍。

<img src={explore} width="400px" />

单击书籍条目可以进入书籍详情，你可以点击目录中的章节进行预览，觉得 OK 就可以加入到书架之中。
