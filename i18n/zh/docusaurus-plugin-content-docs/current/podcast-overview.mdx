## 功能特性

### 被动同步

你可以在设置页面设置播客的同步间隔，但即便设置了同步间隔，也需要你打开播客界面后才会开始同步，而不是每天定时定点地进行后台同步。

### iTunes 搜索

应用内置了基于 iTunes API 的播客探索模块，有 17 个子分区，你也可以搜索自己感兴趣的播客进行订阅。

<img src={explore} />

由于不同国家和地区显示的内容应该不同，应用尊重当前系统的区域设置，也就是说，你在中国就不会给你显示美国的内容。

当然，你可以在 Windows 设置中调整当前区域，依次打开：

`系统设置` -> `时间和语言` -> `国家或地区`，然后调整为你预期的国家或地区即可。

:::tip

调整完区域设置后最好重启一下应用。

:::
